快速入门

Features

加速开发

像原生标签一样使用组件；初始化和复杂逻辑封装到组件定义中；每个组件都有完整的生命周期；动态更新数据和UI渲染；丰富的命令行

拥抱标准

设计理念拥抱Web Components，解决标准落地的兼容性问题；Custom Elements作为组件开发、使用规则；部分提供Shadow Dom功能；完全支持Templating

兼容性广

PC: Chrome; Firefox; IE8~11; Safari

Mobile: Android>=2.3; IOS>=5; UC、QQ、百度等三方手机浏览器

开发生态

基础提供基于BoostUI封装的Elements；三方组件接入生态，同标准下共享开发资源

核心功能

注册机制

提供新Element类的注册接口 注册后就拥有Element的基础接口和生命周期

完整生命周期

和WebComponents行为和名称一致的生命周期，created、attached、attributeChanged、dettached

声明式使用

原生tag一样使用新Element；声明式的设置属性
<r-slider aaa="test" > </r-slider>

Element内外隔离

统一接口形式和生命周期事件，隔离内外功能耦合

content标签隔离内外嵌套

部分的css shadow，隔离内部对外部的css污染

灵活的模版语法

{}隔离模版语法和外部的html，模版语法采用原生的js灵活低成本，js内以jsx语法嵌套html

事件、数据和DOM

Element定义内部共享作用域，方便快速的事件绑定和变量使用；提供ref迅速获取DOM和Element实例，省去复杂代码

四步Run项目

配置环境
$ npm install fis3 -g
$ mkdir demo
$ cd demo
$ fis3 init rosetta
安装脚手架
运行Server
$ fis3 release
$ fis3 server install server-env
$ fis3 server start --rewrite --type php
$ fis3 init element
